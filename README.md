# IP tunnel with SSH (ssh-vpn)

This project contains files for creating permanent ip tunnel (vpn) with openssh package.
You can use it between your own hosts.

Benefits of this solution:

1. You have not need any additional software. You just use standart SSH
package which is installed on any linux.

2. Your traffic looks like as trivial ssh-generated one.

WARNING! This feature will work with root access. Don't use
this keys for standart ssh login.
